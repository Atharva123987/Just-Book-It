# 🎬 Just Book It

An AI-powered movie booking application that allows users to search movies, explore showtimes, and book tickets using natural language queries.

Instead of navigating complex filters, users can simply type:
- "Show me Batman movies near me"
- "Book 2 tickets for the 3PM IMAX show"
- "Find all movies running today"

and the system intelligently understands intent, fills missing details, and guides them through booking.

---

## 🚀 Features

- 🤖 **Natural Language Search**  
  Query movies, shows, and bookings conversationally

- 🧠 **Intent Classification & Slot Filling**  
  Handles incomplete or incorrect inputs (movie name, location, show type)

- 🎟️ **Real-time Show Availability**  
  Browse show timings with formats like 2D, 3D, IMAX

- 💺 **Seat Selection & Booking**  
  End-to-end ticket booking flow

- ⚡ **Conversational UI**  
  Quick prompts + fallback handling for unsupported queries

---

## 🏗️ Architecture

This project is split into 3 main components:

- **Frontend (React Native)**  
  User interface and chat-based interaction

- **Backend (Golang)**  
  API layer for movies, shows, and bookings

- **AI Service (Python)**  
  Handles intent detection, entity extraction, and response structuring using LLMs
